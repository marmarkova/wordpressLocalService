За стартирането на тази програма стартирате<br/>
• Wordpress. Може да го изтеглите от https://bitnami.com/stack/wordpress/virtual-machine.</br>
• Windows. </br>
Използвани инструменти:</br>
• Bitnami,</br>
• Wordpress,</br>
• Elementor,</br>
• Customizer Export/Import plugin,</br>
• Local Services Wireframe 3 theme.</br>
Предварителни изисквания.</br>
• Wordpress. Може да го изтеглите от https://bitnami.com/stack/wordpress/virtual-machine</br>
• Elementor. Трябва да имате инсталиран https://wordpress.org/plugins/elementor/.</br>
• Local Services Wireframe 3 theme Може да свалите от Wordpress => Templates => Kit Library.</br>
• Customizer Export/Import. Може да свалите от https://wordpress.org/plugins/customizer-export-import/.</br>
• Windows.</br>
За да стартирате Сайта Wordpress => Appearance => Customize => Export/Import => Choose File => Import. Избираме hello-elementor-export.dat файл.</br>
